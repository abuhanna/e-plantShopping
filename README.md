# e-plantShopping

Welcome to **e-plantShopping** (Paradise Nursery), a premium and responsive e-commerce web application for plant lovers.

## Project Overview

Paradise Nursery is an online store where users can browse a curated selection of house plants, learn about them, and purchase them. The application is built using React, Redux Toolkit, and Vite.

## Features

- **Product Catalog**: Browse plants grouped by categories with prices and descriptions.
- **Cart Management**: Add plants to cart, adjust quantities, see real-time price updates, and remove items.
- **State Management**: Uses Redux Toolkit for seamless cart state management.
- **Responsive Layout**: Elegant user experience on all devices.
